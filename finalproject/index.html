Boston Bruins 2013 Stanley Cup playoff run

  • Blog
  • Contact

There almost wasn't a 2012-13 NHL season, but luckily for hockey fans, the league's owners and Players' Association reached an agreement on January 6, 2013 to save it from being completely wiped out. A 48-game condensed schedule was released instead of the normal 82 and the Eastern and Western Conferences didn't face each other unless they were lucky enough to reach the Stanley Cup Final. Here's everything you need to know about the Boston Bruins Stanley Cup playoff run where they fell to the Chicago Blackhawks, dropping Game 6 of the best-of-7 series, 3-2 after taking the lead at the 12:11 mark of the third period, only to blow it in the final 76 seconds.

Eastern Conference first round

Bruins vs. Maple Leafs

(Bruins win series, 4-3)

  • Game 1: Bruins 4, Toronto 1
  • Game 2: Toronto 4, Bruins 2
  • Game 3: Bruins 5, Toronto 2
  • Game 4: Bruins 4, Toronto 3 (OT)
  • Game 5: Toronto 2, Bruins 1
  • Game 6: Toronto 2, Bruins 1
  • Game 7: Bruins 5, Toronto 4 (OT)

Eastern Conference semifinal

Bruins vs. New York Rangers

(Bruins win series, 4-1)

  • Game 1: Bruins 3, New York 2 (OT)
  • Game 2: Bruins 5, New York 2
  • Game 3: Bruins 2, New York 1
  • Game 4: New York 4, Bruins 3 (OT)
  • Game 5: Bruins 3, New York 1

Eastern Conference final

Bruins vs. Penguins

(Bruins win series, 4-0)

  • Game 1: Bruins 3, Pittsburgh 0
  • Game 2: Bruins 6, Pittsburgh 1
  • Game 3: Bruins 2, Pittsburgh 1 (2OT)
  • Game 4: Bruins 1, Pittsburgh 0

Stanley Cup Final

Bruins vs. Blackhawks

(Blackhawks win series, 4-2)

  • Game 1: Chicago 4, Bruins 3 (3OT)
  • Game 2: Bruins 2, Chicago 1 (OT)
  • Game 3: Bruins 2, Chicago 0
  • Game 4: Chicago 6, Bruins 5 (OT)
  • Game 5: Chicago 3, Bruins 1
  • Game 6: Chicago 3, Bruins 2
